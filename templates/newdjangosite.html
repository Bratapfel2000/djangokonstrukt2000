Instructions to Create a new Django Site


If you want to start a new Project:

0.) Project name shall be "newdjango"

1.) Open Windows Power Shell Admin Mode (PSA) and go to folder where project should be created

2.) Create Project: Enter 'django-admin startproject newdjango'


For a new Site start here:

3.0) If project is already available and you only want to create a new App, start here. App name shall be "newdjangosite"

3.1) Create App: Open PSA and go to project folder. Enter: 'python manage.py startapp newdjangosite'

4.) Show directory in PSA: Enter ls. Your new App should be listed now.

5.) Add App in settings. Open settings.py in main project folder and add 'newdjangosite' to INSTALLED_APPS = [....,...] (don't forget the comma)

6.1) Create view for target url by adding httpresonse
go to newdjangosite/views.py and add 'from django.http import HttpResponse'

6.2) create view itself with function example: def newdjangositeView(request): return render(request, 'newdjangosite.html')
or for a simple test 'return HttpResponse("Hallo")'

8.) add request to urls.py
path('newdjangosite/', newdjangositeView),

9.) add path to views in urls.py:
'from newdjangosite.views import newdjangositeView'

10.) create the newdjangosite.html go to 'templates' and create file 'newdjangosite.html', write something for testing, for example <h1>Hello</h1>
